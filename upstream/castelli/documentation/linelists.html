according to the Kurucz file gfnew.readme

These linelists are used by SYNTHE for synthetic spectra computations.





 nltelines.asc --is the Kurucz line list "nltelines.asc" from his CD-ROM No1 

 nltelines.bin--is the binary version of nltelines.asc 
the file nltelines.asc contains lines of HI, HeI, HeII, CI, CII, OI, NaI, MgI, MgII, AlI, SiI, SiII,
KI, CaI, CaII. 

The Kurucz code to convert nltelines from ascii to binary is: nltelinesasctobin.for


 lowlines.bin -- is the Kurucz atomic line list  taken from http://kurucz.harvard.edu/linelist/linescd
This line list is used both by ATLAS12 and by SYNTHE when it computes a 
spectrum including lines  from predicted levels. This file contains lines of 
all elements up to first 5 stages of ionization. It does not include
lines of HI, HeI, HeII, CI, CII, OI, NaI, MgI, MgII, AlI, SiI, SiII, KI, CaI, 
CaII.

 hilines.bin -- is the Kurucz atomic line list "highlines.bin" taken from 
http://kurucz.harvard.edu/linelist/linescd

This line list is used both by ATLAS12 and by SYNTHE when it computes a 
spectrum including lines from predicted levels. 
This file contains lines of elements up through Zn in stages of ionization 6 
and higher 
    
 diatomicsiwl.bin --corresponds to the Kurucz molecular line list DIATOMIC.DAT from his CD-ROM No 15.  

 schwenke.bin --are the TiO lines from Schwenke (1998) taken from the Kurucz "molecules/TiO" directory  

 eschwenke.bin --are the TiO energy levels derived from the .ASC files available in Kurucz "molecules/TiO" directory 

 h2ofast.bin --are the H2O lines from Partridge & Schwenke (1997) taken from the Kurucz "molecules/H2O" directory
